Detailed Description

Main application file for controlling a homebrew system.

This file contains the setup and loop functions along with the task creation for running various parts of the homebrew system. The tasks are pinned to specific cores of the ESP32 to ensure efficient processing.

◆ setup()

void setup ( )

Setup function.

This function is called once at the beginning to configure the microcontroller. It initializes serial communication, sets up I2C, configures pin modes, and initializes various components such as the heater, PT100, HX71708, and others.

The function also creates several tasks pinned to specific cores, which are responsible for different operations in the homebrew system.
