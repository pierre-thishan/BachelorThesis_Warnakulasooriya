float read_internal_temp ( void )

Reads the internal temperature of the MCU.

This function configures the internal temperature sensor, starts it, reads the temperature in Celsius, and then stops the sensor. The temperature is returned as a floating-point value.

The internal temperature in Celsius.

Task4code function.

Task 4 code.

This function is responsible for handling the "KeepPressing" routine with a timer. It checks the bKeepPressingFlag, and if set, it initiates the KEEP_PRESSING routine, reads the MCU's internal temperature, and waits for a specified delay before repeating. The task runs indefinitely in a loop.
